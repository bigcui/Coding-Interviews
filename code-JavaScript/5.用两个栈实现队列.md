# 5.用两个栈实现队列

## 题目描述
用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。

## 解题思路
**栈：** 先入后出

**队列：** 后入先出

**入栈：** 直接入栈A

**出栈：**
- 如果栈B为空，从栈A出，入栈B（栈A pop，栈B push）；
- 如果栈B不为空，直接从栈B出栈

## Code
```javascript
var inStack = [],
    outStack = [];
function push(node) {
    // write code here
    inStack.push(node);
}
function pop() {
    // write code here
    if (!outStack.length) {
        while (inStack.length) {
            outStack.push(inStack.pop());
        }
    }
    return outStack.pop();
}
```
